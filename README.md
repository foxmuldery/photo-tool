# 老y照片边框生成器

一个0编程基础的小白研发的简单实用的在线照片边框和文字备注生成工具，支持自定义文字和多种边框样式。

## 主要功能

### 边框设置
- 窄边框 (5%)
- 适中边框 (12%)
- 宽边框 (20%)

### 字体选择
- 霞鹜文楷
- 思源黑体
- 思源宋体
- 长仓体
- 马善政楷书

### 字号设置
- 小字 (16px)
- 中字 (24px)
- 大字 (32px)

### 其他功能
- 照片导航（上一张/下一张）
- 文字位置调整
- 图片位置调整

## 使用说明

1. 上传照片
2. 选择边框宽度
3. 选择字体样式
4. 调整文字大小
5. 调整位置
6. 下载成品
